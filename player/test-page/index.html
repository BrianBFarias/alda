IMPORTANT: For alda.wasm to work on this page:

  1. alda.wasm must be present in the same directory.
  2. You must view this page served on localhost, instead of viewing the HTML file directly in your browser.

The easiest way to get this test page working is to run the serve script in this directory. The script will fetch the latest version of alda.wasm and use the npm http-server package to serve the contents of this directory, including this page.
